This SELECT query determines the data that will be indexed.

The query must SELECT the following bindings:

  • ?documentID determines the identifier of the indexed document.
  • ?documentText determines the document content that will be indexed.
  • ?subject
  • ?predicate
  • ?object
This query is executed during index creation to obtain the actual data for the index. Incremental updates are not supported. When data in your repo changes you should rebuild the index. It is a subquery of a more complicated query you can see from the 'View Index Query' link bellow.
